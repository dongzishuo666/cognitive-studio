# 多AI协作认知增强系统 | AI时代的Windows
> 认知增强工作室 | Cognitive Enhancement Studio

## 🎯 项目简介
本项目是面向下一代人工智能的**多智能体协作中枢平台，定位为AI时代的Windows系统**。
它突破了单AI对话的局限，通过多模型并发协作、结构化认知沉淀，将各类大模型、专业AI、工具能力统一纳入体系，形成强大的集体思考能力，最终为用户实现**认知增强、决策增强、创造力增强**，大幅提升个人与组织解决复杂问题的能力。

如同Windows统一管理硬件与软件、让人人能用电脑，这套系统让人人都能轻松驾驭多AI集群，用系统化、工程化的方式放大人类智慧，是通往AGI时代的轻量化底层基础设施。

---

## ✨ 核心亮点
### 1. 创新的多AI协作模式
- 支持DeepSeek、Gemini、Claude等多款主流大模型**并发/轮流发言**
- 内置「正反辩论」「角色扮演」等13种协作模式，模拟多元视角
- 比单一AI回复更有深度和广度，完美适配学术思辨、商业决策推演等复杂场景

### 2. 结构化+记忆化的认知沉淀
- 12部分结构化总结（核心摘要、行动建议等），将碎片化讨论转化为体系化报告
- 长期记忆功能，后续讨论可基于历史框架迭代，实现「认知连续性」
- 彻底解决传统AI工具「单次对话、无沉淀」的痛点

### 3. 隐私与轻量化优势
- 纯前端架构+本地存储，API Key、记忆数据仅存于浏览器，无服务器泄露风险
- 零后端、零部署成本，保护用户隐私的同时，保证使用便捷性

### 4. 场景化适配性强
- 针对学术综述、商业决策、个人成长等不同场景提供明确的模式推荐
- 降低用户使用门槛，快速落地到具体需求中

## ⚙️ 部署与使用
### 前置要求
- 本地HTTP服务器（不可直接双击HTML打开）
- 各模型API Key（部分模型需代理）

### 部署步骤
1. 克隆本仓库到本地
2. 启动本地HTTP服务器（如`python -m http.server`）
3. 访问对应地址，在配置面板填写各模型API Key
4. 选择协作模式，开始多AI协同思考
